# ЧАСТЬ I: ВИДЕНИЕ И СТОЛПЫ

> Status: DRAFT | Last Updated: 2026-03-17 | Open Questions: — | Decisions Log: —

## 1.1 Pitch

> **Roguelite-головоломка: двумя инструментами охоться на скрытых существ пяти видов в пространстве эфира. Каждая ошибка с выбором инструмента — штраф. Каждый правильный скан — ресурс. Каждое существо — своя угроза.**

## 1.2 Сеттинг и лор

Игрок — один из немногих людей с уникальной способностью входить в **пространство Эфира**: параллельное измерение, открытое Институтом. Там обитают **Эфемеры** — существа пяти видов, скрытые в сетке пространства.

Три организации используют игрока для своих целей:
- **Институт** — научная организация, изучает Эфир и его обитателей.
- **Госпиталь** — военная структура, охотится за биологическими образцами.
- **Чёрный рынок** — конгломерат промышленных разведок, скупает всё.

## 1.3 Design Pillars

### Pillar 1: «Правильный инструмент — ключ»
Locator на Эфемере = ошибка с двойными последствиями (штраф + потеря ресурса). Echobeamer на Эфемере = правильно.

### Pillar 2: «Каждый цвет — своя угроза»
Пять типов Эфемеров требуют разной тактики. Зелёный — безопасен, но медленен. Жёлтый — взрывной. Красный — тикающая бомба. Синий — убегает. Фиолетовый — ломает правила.

### Pillar 3: «Организации работают на тебя и против тебя»
Чем больше исследуешь → тем лучше информация от Института перед следующим забегом. Госпиталь даёт квесты и лечит. Чёрный рынок — непредсказуемый рынок с меняющимися ценами.

### Pillar 4: «Числа — единственная карта»
Resonance Numbers — единственный источник информации. Мастерство = читать числа, не ошибаться с инструментом.

## 1.4 Тон

**Визуальный:** Тёмный океан-Эфир. Сонарные волны. Существа светятся своим цветом.

**Аудио:** Атмосфера подводных лодок и китовых криков. Locator — «пинг». Echobeamer — «тум». Штраф — резкий сбой. Фон — ambient underwater + синтетические китовые крики.

## 1.5 Матрица вдохновений

| Источник | Что берём | Что отвергаем |
|----------|-----------|---------------|
| Minesweeper | Сетка, числа, логика дедукции | Мины как смерть, бинарный исход |
| Эхонавтика 2.0 | Двойные инструменты, мембраны, экономика ресурсов | Физический компонент, 2–4 игрока |
| Slay the Spire | Структура по актам, выбор пути | Колода карт |
| Subnautica | Атмосфера исследования неизвестного | 3D-action |

# ЧАСТЬ II: ЯДРО ГЕЙМПЛЕЯ

> Status: DRAFT | Last Updated: 2026-03-17 | Open Questions: Q1, Q2, Q4 | Decisions Log: —

## 2.1 Сквозная цепочка

```
Числа → Дедукция → Инструмент → Результат → Ресурсы → Организации → Следующий забег
```

## 2.2 Выбор инструмента (ключевая механика)

| Клетка | Locator (бесплатно) | Echobeamer (платно) |
|--------|--------------------|--------------------|
| Пустая (число = 0) | ✓ +1 энергия в батарею | Трата, нет пользы |
| Числовая (число ≥ 1) | ✓ 0 энергии; число раскрыто | Трата, нет пользы |
| Сегмент Эфемера | ✗ ШТРАФ + сегмент **открывается навсегда** (Echobeamer невозможен) | ✓ Правильный скан; ресурс; спецэффект цвета |
| Мембрана (не последняя) | ✗ ШТРАФ + мембрана **открывается навсегда** (Trigger потерян) | ✓ Скан; Trigger не срабатывает |
| Мембрана (ПОСЛЕДНЯЯ) | ✗ ШТРАФ + Trigger потерян | ✓ **MEMBRANE TRIGGER — бонус!** |

> **Ключевое правило (v0.3):** Сегмент, раскрытый Locator-ом, является **«открытым»** — он виден, но **Echobeamer и любой другой инструмент его просканировать не может**. Ресурс с него недоступен. Если «открыт» мембранный сегмент — Membrane Trigger потерян навсегда для этого Эфемера.

## 2.3 Анатомия одного хода

1. Читаю числа → строю картину расположения Эфемеров
2. Дедукция: пустое (Locator) или Эфемер (Echobeamer)?
3. Применяю инструмент → результат + обновление сетки
4. Проверка цветовых эффектов (агрессивное состояние, страх, инверсия)
5. Проверка: мембрана доступна как последний скан?
6. Проверка батареи: перегрузка? Конец комнаты?

## 2.4 Условия конца и победы

**Конец комнаты:** Энергия = 0 И нет нераскрытых клеток, из которых Locator дал бы энергию.

**Membrane Trigger:** Бонус за правильный порядок (все сегменты Echobeamer-ом → мембрана последней). Не обязателен для завершения комнаты. Стандартная награда в прототипе: **+2 энергии + 3 ОИ + 10 монет** (финальные бонусы конкретных типов мембран M-01..M-12 определяются дизайном полной версии).

**Поражение в забеге:** HP = 0 (от штрафов, перегрузки батареи, цветовых эффектов).

**Permadeath:** Да. **Что сохраняется между забегами:** только система разблокировок — открытые карты эксклюзивного оборудования организаций (H/I/BM) и разблокированные фракции (Harvesters, Resonators, Mappers) с их оборудованием. Деньги, ОИ, ресурсы, снаряжение — **сбрасываются** при каждом новом забеге.

## 2.5 Режимы сложности

| Режим | HP | Батарея (ёмк./старт) |
|-------|----|----------------------|
| Casual | 5 | 10 / 5 |
| Standard | 3 | 7 / 3 |
| Expert | 2 | 6 / 2 |
| Deep Echo | 2 | 5 / 2 |

# ЧАСТЬ III: МЕХАНИКА СЕТКИ

> Status: DRAFT | Last Updated: 2026-03-17 | Open Questions: — | Decisions Log: —

## 3.1 Типы клеток

| Тип | Символ | Описание |
|-----|--------|----------|
| Нераскрытая | `░` | Скрыто |
| Пустая | `·` | Число = 0; авторасрытие flood fill |
| Числовая | `1–8` | N сегментов Эфемеров в 8 соседях |
| Сегмент Эфемера | `▓` (цвет по типу) | Требует Echobeamer; Locator = штраф + «открыт» |
| Мембрана | `M` | Фиксированный сегмент; Trigger если последний Echobeamer |
| Открытый сегмент | `○` | Раскрыт Locator-ом; недоступен для Echobeamer |
| Исследованный | `✓` | Просканирован Echobeamer-ом |

## 3.2 Resonance Number System

Число = кол-во сегментов тел Эфемеров среди 8 соседей.

| Число | Тактика |
|-------|---------|
| 0 (пустая) | Locator → +1 энергия; auto-reveal flood fill |
| 1–2 | Аккуратная дедукция |
| 3–5 | Скорее всего рядом крупный Эфемер → Echobeamer |
| 6–8 | Несколько Эфемеров пересекаются |

**Числовая клетка (число ≥ 1) = 0 энергии при Locator** (исправление v0.3). Энергия только от пустых (0).

## 3.3 Auto-reveal

Locator/Echobeamer на пустую клетку → flood fill раскрытия связных пустых + числовых соседей. Не пересекает Distortion Area.

# ЧАСТЬ IV: ИНСТРУМЕНТЫ И ДЕЙСТВИЯ

> Status: DRAFT | Last Updated: 2026-03-17 | Open Questions: Q2 | Decisions Log: Locator обновлён: числовая = 0 энергии; сегмент = открыт навсегда.

## 4.1 Locator

**Бесплатный инструмент разведки.**

| Клетка | Энергия | Дополнительный эффект |
|--------|---------|----------------------|
| Пустая (число = 0) | +1 | Flood fill авторасрытие |
| Числовая (число ≥ 1) | 0 | Число раскрыто |
| Сегмент Эфемера (любой цвет) | 0 | ШТРАФ + сегмент открыт навсегда (Echobeamer невозможен) + спецэффект цвета (см. Часть VI) |
| Мембрана | 0 | ШТРАФ + мембрана открыта навсегда (Trigger потерян) |

> **Визуализация прогресса ОИ:** На Locator-е 5 лампочек. Каждые 5 числовых клеток (число ≥ 1), раскрытых Locator-ом, загорается одна. Когда все 5 — +1 Очко исследований, счётчик сбрасывается.

**Апгрейды:**
- *Echo Locator* — на пустой клетке: авторасрытие + 1 случайный сосед дополнительно
- *Deep Scan* — показывает тип скрытой клетки (тело / пусто) до применения

## 4.2 Echobeamer

**Платный инструмент (2 энергии базово).**

| Клетка | Результат |
|--------|-----------|
| Пустая / числовая | Трата энергии; нет пользы; иногда штраф (Q2) |
| Сегмент тела (любой цвет) | Скан + спецэффект цвета (ресурс/состояние) |
| Мембрана (не последняя) | Скан без Trigger |
| Мембрана (ПОСЛЕДНЯЯ) | **Membrane Trigger!** |
| Открытый сегмент | **Невозможно** |
| Distortion Area | ×2 стоимость (4 энергии) |

## 4.3 Гарпун (Harpoon)

**Инструмент Госпиталя; особая роль — захват.**

- **Стоимость:** 5 энергии
- **Эффект:** Скан сегмента. Если после применения у Эфемера остаётся **менее 3 неисследованных сегментов** — они автоматически закрашиваются как исследованные.
- **Квест-бонус:** При наличии активного квеста Госпиталя на этого Эфемера → награда + анимация.
- **Ограничение:** Гарпун не активирует Membrane Trigger (автозакрашивание ≠ правильный порядок).

## 4.4 Сеть (Net)

**Расходник Госпиталя; одноразовый.**

- **Стоимость:** 0 энергии
- **Эффект:** Если сетью исследован **последний** сегмент Эфемера → квест-бонус (при наличии квеста).
- **Первая сеть:** Выдаётся бесплатно после первого лечения в Госпитале.
- **Занимает:** 1 слот инвентаря. Одноразовая (убирается после применения).
- **Сеть НЕ активирует Membrane Trigger** — она не является Echobeamer.

## 4.5 Sonar Charges (Расходники)

| Расходник | Описание | Источник |
|-----------|----------|---------|
| Deep Ping | Раскрывает числа в 5×5 (без штрафа, не «открывает» сегменты) | Старт + рынок |
| Resonance Shield | Следующий штраф отменён | Рынок |
| Повербанк | +3 энергии в батарею; 1 слот | Чёрный рынок / Институт (за жёлтый эфир) |
| Analysis Burst | Показывает тип каждой клетки в 3×3 до применения | Рынок |

## 4.6 Action Economy

- Минимум за ход: 1 действие
- Locator: не ограничен по количеству
- Echobeamer: не ограничен (пока есть энергия)
- Все инструменты применяются только к нераскрытым клеткам (кроме особых)

# ЧАСТЬ V: ЭКОНОМИКА РЕСУРСОВ

> Status: DRAFT | Last Updated: 2026-03-17 | Open Questions: Q10 | Decisions Log: 7 типов ресурсов: 3 универсальных + 5 от Эфемеров.

## 5.1 Схема ресурсов

```
УНИВЕРСАЛЬНЫЕ:
  Энергия (батарея) ──── тратится на инструменты в ходе сессии
  Деньги ─────────────── тратятся в Госпитале и на Чёрном рынке
  Очки исследований ─── тратятся в Институте (База знаний)

РЕСУРСЫ ЭФЕМЕРОВ:
  Зелёная эссенция ─── Госпиталь (HP)
  Жёлтый эфир ──────── Институт (батарея)
  Красный жемчуг ────── любая организация (эксклюзивное оборудование)
  Кристалл ксилла ───── конвертируется в 10 ОИ (или захватывается)
  Эссенция варпа ─────── боссы (отмена хода); не продаётся
```

## 5.2 Ресурс 1: Энергия (Батарея)

| Параметр | Standard | Casual | Expert |
|----------|----------|--------|--------|
| Ёмкость | **6** | 10 | 5 |
| Старт | 3 | 5 | 2 |
| Перегрузка | –1 HP | –1 HP | –1 HP |

> **v0.7:** Standard ёмкость снижена с 7 до 6 по результатам плейтеста (слишком мало давления на батарею при 7). Expert — с 6 до 5 соответственно.

**Источники:** Locator на пустой клетке (+1); Membrane Trigger энерг. типа (+X); Повербанк (+3); жёлтый Echobeamer (+4 или до лимита).

**Расход:** Echobeamer (2); Echobeamer в Distortion (4); Гарпун (5); некоторые расходники.

**Конец комнаты:** Энергия = 0 + нет пустых нераскрытых клеток.

# ЧАСТЬ VI: ЭФЕМЕРЫ — 5 ЦВЕТОВ

> Status: DRAFT | Last Updated: 2026-03-17 | Open Questions: Q3, Q5, Q6, Q7, Q8 | Decisions Log: Интегрированы из Vid_color_ef.docx.

## 6.1 Общие правила

- **Мембрана:** Фиксированный сегмент (позиция неизменна). Membrane Trigger ТОЛЬКО если просканирован Echobeamer-ом последним среди всех сегментов.
- **Открытый сегмент:** Если сегмент раскрыт Locator-ом — «открыт» навсегда; Echobeamer невозможен; ресурс потерян.
- **Цветовые спецэффекты:** Применяются при любом обнаружении сегмента (Locator или Echobeamer), если не указано иное.

## 6.2 Зелёные / Добрые / Пассивные

**Визуал:** Зелёные сегменты. Спокойное свечение.

**Размер:** 4–7 сегментов.

| Инструмент | Эффект |
|-----------|--------|
| Locator | Штраф + сегмент открыт навсегда. **Нет доп. спецэффекта.** *(Q-A: стандартный штраф? Или нет штрафа?)* |
| Echobeamer | **+1 Зелёная эссенция** |

**Зелёная эссенция → Госпиталь:**

| Трата | Эффект |
|-------|--------|
| 3 зелёные эссенции | Восстановить 1 HP |
| 5 / 7 / 10 зелёных эссенций | Увеличить лимит HP: +1 / +2 / +3 |

**Тактика:** Самый безопасный тип. Приоритет — запас эссенций для Госпиталя.

## 6.3 Жёлтые / Опасные / Взрывные

**Визуал:** Жёлтые сегменты. Пульсирующее свечение.

**Размер:** 3–4 сегмента.

| Инструмент | Эффект |
|-----------|--------|
| Locator | Штраф + **ВЗРЫВ:** раскрывает все клетки 3×3 вокруг + 2 энергии. Если батарея полна — перегрузка + доп. –HP. Сегмент открыт. |
| Echobeamer | **+4 энергии** (если лимит позволяет). Если нет — батарея до максимума без перегрузки + **+1 Жёлтый эфир** |

> **Сомнение D-1:** +4 энергии от Жёлтого при ёмкости батареи ~7 и старте ~3 = почти полная батарея за одно действие. Может быть слишком мощно в начале. Предлагаемая альтернатива: **+2 энергии + 1 Жёлтый эфир** (стандартно) и только при полной батарее: +1 Жёлтый эфир без энергии. Оставляю как есть до плейтеста.

**Жёлтый эфир → Институт:**

| Трата | Эффект |
|-------|--------|
| 3 жёлтых эфира | Купить Повербанк (3 заряда, 1 слот) |
| 5 / 7 / 10 жёлтых эфиров | Увеличить лимит батареи: +1 / +2 / +3 |

**Тактика:** Locator смертелен — взрыв вскрывает зону и может каскадом открыть других Эфемеров. Echobeamer выгоден. Быстро завершать.

## 6.4 Красные / Агрессивные / Злые

**Визуал:** Красные сегменты. Угрожающая пульсация.

**Размер:** 3–5 сегментов.

| Инструмент | Эффект |
|-----------|--------|
| Locator | Штраф + **агрессивное состояние**: каждый 3-й ход –1 HP, пока Эфемер не исследован полностью. Сегмент открыт. |
| Echobeamer | **Агрессивное состояние**: каждый 3-й ход –1 HP, пока не исследован полностью. С 30% вероятностью: **+1 Красный жемчуг** |

> **Замечание:** Агрессивное состояние запускается ЛЮБЫМ инструментом. Избежать его нельзя — только завершить сканирование быстро.

> **Решение D-2 / Q-D (ЗАКРЫТО):** 1 ход = 1 применение любого инструмента.

**Красный жемчуг → любая организация:**

| Трата | Эффект |
|-------|--------|
| Продать 1 КЖ в любой орг. | 20 монет |
| Продать 1 КЖ на Чёрном рынке | 40 монет (вдвое выше) |
| 5 КЖ → разблокировка | 1 случайный незакрытый эксклюзив данной организации становится доступным в магазине навсегда |
| Купить 1 КЖ на Чёрном рынке | 100 монет |

**Тактика:** Опасно, но прибыльно. Красные Эфемеры дают редкий ресурс. Приоритет — завершить быстро до следующего тика агрессии.

## 6.5 Синие / Трусливые / Убегающие

**Визуал:** Синие сегменты. Дрожащее свечение.

**Размер:** 4–7 сегментов.

| Инструмент | Эффект |
|-----------|--------|
| Locator | Штраф + **состояние страха**: через **6 ходов**, если Эфемер не исследован полностью — он исчезает. Игрок теряет 5 произвольных единиц ресурсов. Сегмент открыт. |
| Echobeamer | **50%:** состояние страха (те же последствия). **20%:** +1 **Кристалл ксилла**. Оба эффекта независимы. |

> **Решение Q-F:** Таймер страха = **6 ходов** (было 5). Если окажется мало — усилим штраф вместо таймера.

> **Решение D-4 (ЗАКРЫТО):** Штраф = потеря **5 произвольных единиц ресурсов** (монеты, эссенции, эфир, жемчуг — случайный набор). Создаёт напряжение из-за непредсказуемости потери.

**Кристалл ксилла:** Немедленно конвертируется в **+10 ОИ** (если нет BM-06 Резервуара ксиллы). С BM-06: кристалл сохраняется в слоте (до 3 шт.), продаётся в BM за **80м/кристалл** — вместо авто-конверсии.

**Тактика:** Запускает гонку со временем. Как только Locator или Echobeamer (50%) активировал страх — всё бросай и завершай этого Эфемера.

## 6.6 Фиолетовые / Варп / Проклятые

**Визуал:** Фиолетовые сегменты. Искажённое мерцание.

**Размер:** 3–5 сегментов.

| Инструмент | Эффект |
|-----------|--------|
| Locator | Штраф + **инверсия**: пока этот Эфемер не исследован полностью — Locator на пустых клетках (число = 0) **теряет 1 энергию** вместо +1. Визуал: лампочки Locator-а становятся фиолетовыми. Сегмент открыт. |
| Echobeamer | **40%:** случайный ресурс. **40%:** ±1 HP (50/50). **10%:** +1 **Эссенция варпа**. (Сумма = 90%; 10% — ничего.) |

> **Решение D-5 / Q-E (ЗАКРЫТО):** Энергия не уходит ниже 0. При инверсии + батарея = 0 + Locator на пустой → авто-штраф –1 HP.

**Эссенция варпа:** Легендарный ресурс. Применяется только в бою с боссами: **позволяет отменить один сделанный ход**. Не продаётся. Не теряется при смерти.

**Тактика:** Высокий риск с непредсказуемым результатом. Echobeamer нестабилен. Приоритет — завершить как можно быстрее, чтобы снять инверсию.

## 6.7 Система мембран (12 типов, 2 случайных на цвет)

**Правило распределения:** В начале каждого забега случайным образом каждому из 5 цветов назначаются **2 типа мембран** из пула 12. Итого 10 уникальных типов в одном забеге — разнообразие выше.

**Как работает Trigger:** Membrane Trigger срабатывает только если мембранный сегмент просканирован **Echobeamer-ом последним** среди всех сегментов данного Эфемера.

**12 типов мембран (утверждено v0.4):**

| # | Название | Эффект | Примечание |
|---|----------|--------|------------|
| M-01 | Пульс | +3 энергии в батарею. Если батарея переполнена — **+1 к лимиту батареи** вместо перегрузки | Безопасная энергия |
| M-02 | Волна | Раскрыть строку ИЛИ столбец (выбор игрока) | Тактическое раскрытие |
| M-03 | Сонар | Раскрывает все **числовые клетки** в комнате. Показывает позиции **2 не-мембранных сегментов** для 2 нераскрытых Эфемеров (или всех, если нераскрытых ≤ 2) | Сегменты не раскрываются; только разведка |
| M-04 | Усиление | Следующие 3 Echobeamer — бесплатны. **UI:** 2 лампочки на Echobeamer мигают во время действия эффекта | Визуальная индикация обязательна |
| M-05 | Исследование | +3 Очка исследований | Прямая прокачка базы |
| M-06 | Щит | Следующий штраф (Locator на Эфемере) отменён | Страховка |

| M-10 | Резонанс | Следующий Membrane Trigger этого забега активирует свой эффект дважды. **UI:** 2 лампочки на Echobeamer перемигиваются во время действия | Визуальная индикация обязательна |

| M-12 | Регенерация | +1 HP. Если HP уже на максимуме — **+1 к лимиту HP** | Зеркало M-01 для здоровья |

# ЧАСТЬ VII: ОРГАНИЗАЦИИ

> Status: DRAFT | Last Updated: 2026-03-17 | Open Questions: Q9 | Decisions Log: Интегрировано из Vid_org.docx.

## 7.1 Лор организаций

**Институт** — научная организация, открывшая пространство Эфира. Обеспечивает игрока доступом и базовым оборудованием. Цель: сбор данных, создание новой физики.

**Госпиталь** — принадлежит военным. Интересуют биологические материалы для исследований бессрочного продления жизни. Даёт квесты на захват Эфемеров.

**Чёрный рынок** — конгломерат промышленных разведок и частных подрядчиков. Покупает всё, цены нестабильны.

## 7.2 Институт

**Основная польза:** База знаний и разведданные перед сессией.

### Энциклопедия (Knowledge Base)

Все исследованные Эфемеры заносятся в Энциклопедию. Базовая запись (форма + цвет) добавляется автоматически при первом обнаружении. Дополнительные строки покупаются **строго по порядку** — от наиболее ценного к наименее:

| Порядок | Строка информации | Стоимость ОИ |
|---------|------------------|-------------|
| 1 | Тип ресурса | 10 ОИ |
| 2 | Спецэффект Locator | 10 ОИ |
| 3 | Спецэффект Echobeamer | 10 ОИ |
| 4 | Размер (число сегментов) | 10 ОИ |
| 5 | Мембрана 1-я (тип) | 10 ОИ |
| 6 | Мембрана 2-я (тип) | 10 ОИ |

> **Порядок обязателен.** Нельзя купить размер, не зная эффектов. Каждая мембрана раскрывается отдельно.

> **Групповое раскрытие:** Информация относится ко всему цвету — открывается сразу для всех Эфемеров этого цвета в Энциклопедии.

### Статистика в Энциклопедии

Для каждого типа Эфемера Энциклопедия также хранит статистику:
- Сколько встречено всего
- Сколько полностью исследовано
- Сколько исследовано с Membrane Trigger

Статистика видна всегда, даже без покупки строк.

### Разведданные перед сессией

По мере заполнения базы, Институт перед стартом каждой сессии сообщает:
- Сколько Эфемеров в зоне
- Какого цвета / формы
- Какие мембраны у них (если куплено)

Игрок может: подготовить расходники, выбрать оборудование, отказаться от сектора.

### Денежные премии

Институт выплачивает денежную премию за ОИ, собранные **в текущей комнате** (не суммарные за забег):

| ОИ за комнату | Премия |
|---------------|--------|
| < 30 | Нет |
| ≥ 30 (средний объём) | +30 монет |
| ≥ 50 (большой объём) | +45 монет |
| ≥ 70 (феноменальный объём) | +60 монет |

> Премия выплачивается автоматически при завершении комнаты — в конце обратного отсчёта.

### Разблокировка фракции

После достижения **100 суммарных ОИ** разблокируется фракция **Resonators («Разведчики»)**.

## 9.4 Общая механика боссов — Система Глаз

Бой с боссом — принципиально другой контракт:

| Параметр | Обычная комната | Бой с боссом |
|----------|----------------|--------------|
| Противник | Несколько Эфемеров, 3–9 сегм. | Один Мега-Эфемер, 20–60 сегм. |
| Активность | Пассивна (реагирует на ходы игрока) | Действует каждые N ходов независимо |
| Цель | Закрыть комнату (энергия иссякла) | Найти и просканировать все **Глаза** |
| Сетка | Стандартная | Увеличенная, спецструктура |

### Глаза (Boss Membranes)

Каждый босс имеет **2–4 специальных мембраны — Глаза**, расположённых **симметрично** внутри тела. Визуально неотличимы от обычных мембран до момента сканирования.

**Победа = просканировать все Глаза Echobeamer-ом (в любом порядке, если не указано иное).**

Остальные сегменты тела **не обязательны для победы** — они дают ресурсы и затрудняют поиск Глаз через числа. Игрок расчищает ровно столько, сколько нужно для навигации.

Симметрия — механизм честной дедукции: нашёл один Глаз → знаешь примерное положение второго. Resonance Numbers вокруг Глаз образуют характерные паттерны, читаемые с опытом.

### Видимость формы босса

| ОИ перед боссом | Что даёт Институт при входе |
|-----------------|----------------------------|
| < 30 | Только размер сетки. Форма скрыта. |
| 30–80 | **Силуэт** тела (контур без деталей) |
| 80–150 | Силуэт + **оси симметрии** (примерные зоны Глаз) |
| 150+ | Силуэт + **точные позиции** всех Глаз |

> Даже без Института: симметрия + числовые паттерны позволяют найти Глаза дедукцией, просто дольше.

### Тело после победы

После уничтожения последнего Глаза тело становится **инертным**: цветовые эффекты, таймеры и активная механика босса останавливаются. Оставшиеся сегменты можно собрать для ресурсов или покинуть комнату.

### Мembrane Trigger на Глазах

Глаза — это мембраны, поэтому Membrane Reader (E-03) даёт малый бонус при их сканировании. Стандартный M-01..M-12 Trigger на Глазах **не срабатывает** — их эффект определяется типом Глаза (Парализующий / Разъяряющий / Немой / Яростный, см. ниже).

### Гарантированная Organization Room

Перед каждой боссовой комнатой игрок **гарантированно** проходит Organization Room. Это момент конвертации всех накопленных ресурсов в преимущество перед боссом.

### Лут боссов

После победы над боссом — **одна случайная карта оборудования** случайной фракции:

| Босс | Редкость | Ценность продажи (ЧР) | Когда применима |
|------|---------|----------------------|----------------|
| Boss 1 | Обычная | 60–120 монет | Акты 4–6 |
| Boss 2 | Редкая | 200–350 монет | Акты 7–9 |
| Boss 3 | Легендарная | 500–900 монет | Игра завершена |

Для Boss 3: карта показывается в **итоговом экране** как «трофей» (что это было бы, её фракция и эффект). После — только Чёрный рынок, если хочется «дооценить» пройденный забег.

## 9.5 Boss 1: «Медленный Пульс» (Акт 3)

**Тема:** Ритм и дисциплина инструмента. К Act 3 игрок знает только Зелёных и Жёлтых — их он и видит на боссе. Задача — не торопиться с Locator-ом под тихим, но постоянным давлением.

**Сетка:** 12×12. **Сегментов:** 20–24. **Цвета:** Зелёный + Жёлтый. **Глаз: 2** (симметрия лево-право).

### Активная механика: Пульс

Каждые **5 ходов** (таймер виден) — **Resonance Pulse**:
- 2 случайные нераскрытые клетки на теле босса становятся **«враждебными»**
- Враждебные клетки **на сегментах тела** исчезают через 3 хода; клетки **вне тела** остаются враждебными постоянно до завершения боя
- **Locator заблокирован** на враждебной клетке — применить невозможно, ход не тратится
- **Соседство:** использование Locator рядом с враждебной клеткой = **–1 энергия** (не HP)
- Визуал: красное пульсирование. Курсор «запрещено». Соседние клетки — тревожная мерцающая рамка
- **Дополнительное условие поражения:** если все оставшиеся скрытые клетки стали враждебными — выход заблокирован, забег окончен

### Тип Глаз: Парализующие

Оба Глаза одинаковые. Echobeamer на Глаз → **Пульс замирает на 8 ходов** (таймер сбрасывается). Даёт передышку для нахождения второго Глаза без давления.

### Стратегия

Найди первый Глаз (числовые паттерны + симметрия) → Пульс заморожен → спокойно ищи второй.

### Как помогает подготовка

| Ресурс | Бонус |
|--------|-------|
| 80+ ОИ (Институт) | Оси симметрии → примерная зона обоих Глаз |
| HP-лимит 4+ (Госпиталь) | Выдержать 3+ Пульса без критического риска |
| H-03 Анестезия | Паузирует Пульс на 5 ходов |
| Resonance Shield | Поглощает следующий штраф (не защищает от –1 энергии у соседних клеток) |
| Эссенция варпа | Отменить ошибочный ход рядом с враждебной клеткой |

## 9.6 Boss 2: «Хаотический Разряд» (Акт 6)

**Тема:** Давление на батарею при работе со всеми пятью цветами. К Act 6 игрок знает всё. Задача — управлять экономикой под постоянным дренажом и вовремя нейтрализовать цветовые состояния.

**Сетка:** 14×14 (в полной версии — + Distortion Area центральная полоса 4×14). **Сегментов:** 30–38. **Цвета:** все 5. **Глаз: 3–4** (симметрия по вертикальной оси).

> **v0.7:** Имя «Разрядка» → «Хаотический Разряд». Прототип использует 3 Глаза (вертикальная ось симметрии); полная версия — 4 Глаза (2 пары).

### Активная механика: ЭМИ Импульс

Каждые **3 хода** (таймер виден) — срабатывает пульс. С нарастающей вероятностью пульс превращается в **ЭМИ Импульс** вместо генерации враждебных клеток:

| Пульс с момента последнего ЭМИ | Вероятность ЭМИ |
|-------------------------------|-----------------|
| 1-й | 10% |
| 2-й | 30% |
| 3-й | 50% |
| 4-й | 100% |

После любого срабатывания ЭМИ счётчик сбрасывается (следующий пульс снова 10%).

**Эффект ЭМИ Импульса:**
- 50% шанс: **+3 энергии** в батарею. Если перегрузка (выходит за лимит) → –1 HP.
- 50% шанс: **–3 энергии** из батареи. Если энергии меньше 3 → батарея = 0, –1 HP.
- Дополнительно: **один случайный слот карточек (1–3) блокируется на 1 ход**. Слот 0 (Locator) никогда не блокируется.

**Заблокированный слот:** нельзя использовать инструмент/расходник в этом слоте до конца хода. Попытка использования — отказ без траты хода. Блок снимается после первого успешного действия игрока.

> **v0.7 ПРОТОТИП:** ЭМИ Импульс реализован вместо механики Дренажа. Дренаж планируется для полной версии.

### Бывшая механика: Дренаж (планируется в полной версии)

Каждые **4 хода** — **Echo Drain**: –1 энергия из батареи.
- Если батарея = 0 в момент Дренажа → **–1 HP** (тип урона — *дренаж*, не *штраф инструмента*)
- Фаза 2 (после 50% сегментов): Дренаж каждые **2 хода**

### Тип Глаз: Разъяряющие

Все 4 Глаза одинаковые. Echobeamer на Глаз → **Разъярение**: Дренаж каждые 2 хода следующие **4 хода** (ускорение). НО: немедленный бонус **+4 энергии** + ресурс цвета этого сегмента.

**Момент решения:** Сканируй Глаз только при батарее ≥ 5 — иначе Разъярение каскадно опустошит батарею до HP-урона.

### Стратегия

Расчищай тело, выявляй Глаза через паттерны. Перед каждым Глазом — убедись в наполненной батарее. Цветовые состояния (агрессия Красного, страх Синего, инверсия Фиолетового) нейтрализуй перед сканированием Глаза — иначе Разъярение + активное состояние = двойной кризис.

### Как помогает подготовка

| Ресурс | Бонус |
|--------|-------|
| 150+ ОИ (Институт) | Точные позиции всех 4 Глаз + силуэт |
| Расширенная батарея 9–10 | Дренаж и Разъярение не смертельны |
| Повербанки ×2–3 | Экстренная зарядка после Разъярения |
| H-03 Анестезия | Паузирует Дренаж на 5 ходов |
| BM-02 Контрабандный скан | Расчищать Red/Blue/Violet сегменты без цветовых состояний — готовить чистое поле перед Глазами |
| Эссенция варпа | Отменить Глаз, просканированный при пустой батарее |

## 9.7 Boss 3: «Эхоцепь» (Акт 9)

**Тема:** Финал. Всё одновременно. 8 актов подготовки конвертируются здесь.

**Сетка:** 16×16 + 2× Distortion Area (два случайных блока 4×8). **Сегментов:** 50–60. **Цвета:** все 5, по 10–12 каждого. **Глаз: 4** — два типа.

### Два типа Глаз: Немые и Яростные

Все 4 Глаза **визуально неотличимы**. Два — **Немые** (Парализующие), два — **Яростные**.

- **Немой Глаз** (Echobeamer): Chain Resonance паузируется на **6 ходов**. Числовой паттерн вокруг — симметричный, ровный.
- **Яростный Глаз** (Echobeamer): **немедленный** Chain Resonance + ускорение перехода к следующей фазе. Числовой паттерн вокруг — «рваный» (числа скачут сильнее соседей).

Рекомендуемый порядок: сначала Немые (стабилизируют поле), потом Яростные (с позиции контроля). Спектроскоп (I-02) раскрывает тип без сканирования — ключевой инструмент.

### Активная механика: Chain Resonance

Каждые **5 ходов** — **Chain Resonance**: одна нераскрытая клетка рядом с открытым сегментом «заражается» его цветом на 3 хода. Locator на заражённую клетку = эффект этого цвета (включая штраф и состояние).

### Три фазы

| Порог | Фаза | Изменения |
|-------|------|-----------|
| 0–35% сегментов | Фаза 1 | Стандартно. Chain Resonance каждые 5 ходов |
| 35–70% | Фаза 2 | **Глобальная инверсия** (Locator на пустых = –1 энергии) |
| 70–100% | Фаза 3 | Chain Resonance каждые 3 хода; цветовые таймеры ×1.5 быстрее |

Переход фаз — визуальное событие (сетка сотрясается, цвет окружения меняется).

### Стратегия

Фаза 1: Разведка, поиск Немых Глаз. Фаза 2: Инверсия мешает Locator-у — минимизировать использование, переключиться на дедукцию по числам. Фаза 3: Хаос нарастает, но ты уже знаешь где Яростные — бей.

### Как помогает подготовка

| Ресурс | Бонус |
|--------|-------|
| 300+ ОИ (Институт) | Тип каждого Глаза (Немой/Яростный) + точные позиции |
| I-02 Спектроскоп | Определить Немой/Яростный без сканирования прямо во время боя |
| I-03 Полевой анализатор | «Следующий = Глаз?» — навигация к целям в фазе 3 |
| Эссенция варпа ×2–3 | Отменить Яростный в неподходящий момент |
| H-03 Анестезия ×2 | Дважды заморозить Chain Resonance в критический момент |
| 6 слотов инвентаря | Все три организации одновременно |

## 12.1 Правила взаимодействия

**Общие принципы:**
1. Боссы — это Эфемеры. Большинство предметов применяются по стандартным правилам.
